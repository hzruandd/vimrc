使用Vundle，极轻极简的管理和配置Vim，使用只需以下简单几步操作：

安装Vundle：git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

安装插件： :BundleInstall

更新插件： :BundleInstall!

卸载不在列表中的插件： :BundleClean"

更多插件：http://vim-scripts.org/vim/scripts.html
